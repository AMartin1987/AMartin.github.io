Soy una psicóloga argentina incursionando de manera autodidacta en el mundo de la programación, con particular interés en el uso de Python para el desarrollo backend.

Disfruto enormemente de resolver problemas en este ámbito, por lo que mi objetivo es convertir mi pasatiempo en una actividad de tiempo completo.

Mi metodología de aprendizaje involucra estudiar literatura sobre redes y fundamentos de la programación (e.g. el Backend Developer Roadmap) y realizar cursos (e.g., CS50) mientras aplico los conceptos adquiridos en la realización de proyectos que resuelven problemas relevantes para mí.

Hasta el momento he adquirido experiencia en el uso de Flask y SQLite. Cuento además con conocimientos básicos para implementar el frontend (HTML5, CS3, Javascript, JQuery, Bootstrap 5). Actualmente me encuentro trabajando en un nuevo proyecto con el objetivo de aprender Django y PostgreSQL.

PROYECTO REALIZADO

Backend utilizando Flask y SQLite3, con autenticación, dashboard e integración con APIs de Google. Frontend realizado con Bootstrap 5, con funcionalidades escritas con Javascript y JQuery.
